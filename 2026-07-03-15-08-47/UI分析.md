# 小狐搜搜「逐玉」资源弹窗 UI 分析

## 页面概览

网站：小狐搜搜（uuyue.top）是一个网盘资源搜索引擎，当前页面搜索关键词为「逐玉」。整体视觉风格偏向现代、清透的「玻璃拟态」设计：

- **背景**：浅色模式下使用淡蓝白渐变（`#f5f8ff` / `#eaf2ff`），深色模式为深蓝黑（`#0b1020`）。
- **主题色**：靛蓝/蓝紫渐变（`#6366f1` ~ `#2563eb`），用于按钮、强调、状态色。
- **卡片**：白色半透明底 + 细边框 + 柔和投影，统一使用 14~18px 大圆角。

## 「获取资源」按钮

搜索列表右侧触发弹窗的按钮样式如下：

| 属性 | 值 |
|------|-----|
| 类名 | `.quantum-item .go` |
| 尺寸 | 高 28px，内边距 0 12px |
| 圆角 | 10px |
| 默认状态 | 白底（rgba 255,255,255,.82），蓝紫边框（rgba 18,90,255,.22），12px 字重 800 |
| 悬停 | 背景变为主题蓝，文字变白，轻微上浮 1px，阴影加深 |
| 箭头 | `::after` 伪元素插入 `→` |
| 失效态 | 灰底、灰字、禁用光标 |

## 资源弹窗（`.dialogUrlBox` / `.q-res-modal`）

弹窗由 Element Plus 的 `el-dialog` 二次封装，自定义了完整视觉：

### 尺寸与布局

- **宽度**：`clamp(360px, 92vw, 620px)`，移动端保持 92vw。
- **最大高度**：`min(92vh, 860px)`，超出时内部滚动。
- **圆角**：18px，顶部/底部/卡片均使用 14~18px 圆角。
- **内部布局**：桌面端双栏（左侧 220px 媒体区 + 右侧内容区），移动端单栏。

### 视觉风格

- **浅色背景**：线性渐变 `rgba(255,255,255,.98)` → `rgba(245,250,255,.92)`，带 1px 半透明边框。
- **深色背景**：多层径向渐变（天蓝/蓝紫）+ 深蓝黑底 `rgba(10,10,26,.92)`，营造暗夜光晕感。
- **阴影**：`0 26px 72px rgba(2,6,23,.14)`，深度柔和。
- **毛玻璃**：顶部标题栏和底部操作栏均使用 `backdrop-filter: blur(12px)`。

### 顶部标题栏（`.q-res-top`）

- 标题 16px，字重 900，字母间距 0.2px。
- 副标题/更新时间 12px，72% 透明度。
- 关闭按钮 34×34px，12px 圆角，白底半透明，带阴影。

### 左侧媒体区

- **封面**：3:4 比例，16px 圆角，占位灰底，object-fit: cover。
- **二维码**：140×140px 白色容器，18px 圆角，带阴影，下方「扫码获取」提示。

### 右侧内容区

分三个信息卡片（`.q-res-section`）：

1. **资源信息**：两列网格字段（文件大小、更新时间），顶部有网盘类型 pill + 分类 pill。
2. **链接信息**：链接文本框 + 复制按钮；密码文本框 + 复制密码按钮。
3. **资源描述**：最多 4 行截断，13px 行高 1.65。

### 底部区域

- **声明**：12px 小字，居中，带铃铛 emoji，强调「不存储、不传播」。
- **QQ 群链接**：紫色渐变按钮，带外发光，hover 上浮。
- **操作按钮**：「打开链接」（ghost 样式）+「关闭」（蓝紫渐变主按钮）。

### 响应式

- 768px 以下：双栏变单栏，封面与二维码横向排列，二维码缩小到 132px，字段网格变单列。

## 设计亮点

1. **一致性**：所有圆角、间距、字重、透明度均成体系，12px 小号字大量使用于标签/说明。
2. **层次感**：通过半透明底、渐变、阴影、毛玻璃区分弹窗与页面背景。
3. **状态可见**：链接检测中、已失效、正常等状态均有对应文字和颜色提示（`#ef4444` 错误红）。
4. **移动端适配**：媒体区横向排布，充分利用窄屏空间。

## 可借鉴处

- 弹窗宽度使用 `clamp()` 同时兼顾移动端和桌面端。
- 用 `aspect-ratio: 3 / 4` 固定封面比例，避免布局抖动。
- 信息分区用卡片化设计，降低认知负荷。
- 操作按钮固定在底部，符合移动端手势习惯。
